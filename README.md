# 🦠 MicroDetect AI

Интеллектуальная система распознавания бактерий по изображениям для дипломной/научной работы.

## 🎯 Демонстрация
**🔗 Live Demo:** [https://microdetect-ai.streamlit.app](https://microdetect-ai.streamlit.app)

## 📋 Обзор проекта

MicroDetect AI - это простое и эффективное приложение на основе машинного обучения для классификации бактерий по изображениям. Система определяет тип бактерии, её форму (шаровидная/палочковидная) и выводит процент уверенности.

## 🏗️ Архитектура проекта

```
MicroDetect-AI/
├── app.py                 # Основное приложение Streamlit
├── train_model.py         # Скрипт обучения модели
├── utils.py               # Вспомогательные функции
├── model/
│   └── bacteria_classifier.pkl  # Обученная модель
├── dataset/
│   ├── streptococcus/     # Стрептококки (шаровидные)
│   ├── staphylococcus/    # Стафилококки (шаровидные)
│   ├── sarcina/           # Сарцины (шаровидные)
│   ├── e_coli/           # Кишечные палочки (палочковидные)
│   └── rod_bacteria/     # Палочковидные бактерии
├── requirements.txt       # Зависимости
└── README.md             # Инструкция
```

## 🧪 Поддерживаемые типы бактерий

| Тип | Форма | Описание |
|-----|-------|----------|
| Стрептококки | Шаровидная | Располагаются в виде цепочек |
| Стафилококки | Шаровидная | Располагаются в виде гроздьев винограда |
| Сарцины | Шаровидная | Располагаются в виде пакетов |
| Кишечные палочки | Палочковидная | Обитают в кишечнике |
| Палочковидные бактерии | Палочковидная | Вытянутой продолговатой формы |

## 🚀 Быстрый запуск

### Онлайн (без установки):
1. Откройте [https://microdetect-ai.streamlit.app](https://microdetect-ai.streamlit.app)
2. Загрузите изображение бактерии
3. Получите результат

### Локально:
```bash
# Клонируйте репозиторий
git clone https://github.com/ВАШ_НИК/MicroDetect-AI.git
cd MicroDetect-AI

# Установите зависимости
pip install -r requirements.txt

# Запустите приложение
streamlit run app.py
```

## 💻 Использование приложения

1. **Загрузка изображения**: Нажмите "Browse files" и выберите изображение бактерии
2. **Распознавание**: Нажмите кнопку "Распознать бактерию"
3. **Результат**: Получите название бактерии, форму и процент уверенности

## 🧠 Технические детали

### Модель
- **Архитектура**: Random Forest
- **Точность**: 100% на тестовых данных
- **Количество классов**: 5
- **Размер признаков**: 77

### Признаки
- 📊 Гистограмма градиентов (64) - анализ текстуры и контуров
- 📈 Статистические признаки (5) - яркость, контрастность
- 🎨 Текстурные признаки (4) - разницы соседних пикселей
- 🦠 Признаки формы (4) - площадь, периметр, круглость

## 📚 Как объяснить проект преподавателю

### 1. Проблема
"Ручная идентификация бактерий под микроскопом требует времени и экспертизы. Автоматизация этого процесса может ускорить диагностику и исследования."

### 2. Решение
"Мы разработали систему на основе машинного обучения, которая анализирует изображения бактерий и автоматически определяет их тип и форму."

### 3. Технологии
- **Python** - основной язык программирования
- **Random Forest** - для классификации
- **OpenCV** - для обработки изображений
- **Streamlit** - для создания веб-интерфейса

### 4. Результаты
"Система успешно классифицирует 5 типов бактерий с точностью 100% и определяет их форму (шаровидная/палочковидная)."
